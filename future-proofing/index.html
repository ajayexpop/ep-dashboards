Value Assessment Framework

Each story's value is structured: Impact type(s) (Revenue, Retention, Risk Reduction, Adoption, Cost Reduction) + Customer Tier + Reach + Confidence + Evidence. Click any story (a table row, card or roadmap item) to open it and set these. The Value Score = Tier x Reach x Impact x Confidence (Impact = the highest-weight selected impact type; the score is 0 until Tier, Reach, Impact and Confidence are all set). Evidence backs the confidence - a story with no evidence is flagged. Filter the table by impact type, team, pillar, status or text.
Saving to git: press Save & export - your assessments are kept in this browser and a story_savings.json file is downloaded (and copied to your clipboard). Hand that file to me and I commit it to the repository and republish, so the values live at git level and everyone sees them, not just this browser. Reset draft discards local edits and restores the published values.
